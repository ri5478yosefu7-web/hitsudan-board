# 📝筆談ボード

耳が聞こえない方や、音声での会話が難しい場面で利用できる筆談支援ツールです。

## 主な機能

* 定型文ボタンによる会話支援
* お気に入り登録機能
* 定型文検索機能
* スマートフォン対応
* ブラウザだけで利用可能
* データは端末内（LocalStorage）に保存

## 利用方法

1. 定型文ボタンを選択します
2. 必要に応じてお気に入りへ登録します
3. 検索機能で目的の定型文を探します
4. 相手へ画面を見せて利用します
